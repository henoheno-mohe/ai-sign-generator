# neon-sign-maker

屋内向けネオンサイン（個人向け）に特化した「手書き/テキスト → AIで設置イメージ生成 → 問い合わせ獲得」用のフロントエンド（Next.js）です。

## 目的（MVP）

- **ゴール**: まずは **問い合わせ獲得（A）** まで
- **入力**: **テキスト** or **手書き画像アップロード**（単線）
- **出力**: **テンプレ壁（3種）** に合成した **設置イメージ（B）**
- **購入**: STORES等の外部購入ページへ誘導（このアプリ内で決済しない）
- **製造**: 中国工場へ委託（別運用）

## セットアップ

1) 依存関係インストール

```bash
cd neon-sign-maker
npm install
```

2) 環境変数

`.env.local` を作成し、既存プロジェクトと同じキー名で設定してください。

```bash
NEXT_PUBLIC_NANO_BANANA_API_KEY=...
```

3) 起動

```bash
npm run dev
```
